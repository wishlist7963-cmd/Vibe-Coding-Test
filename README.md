# OmniGo - 통합 모빌리티 라이프 플랫폼

OmniGo(옴니고)는 일상의 모든 이동(차량 공유, 택시 호출, 따릉이, 전동 킥보드, 대중교통 환승)을 단 하나의 앱으로 통합하여 최적의 경로를 탐색, 예약, 결제할 수 있는 모빌리티 통합 플랫폼의 랜딩 페이지입니다.

## 🌟 주요 기능
1. **Header & 네비게이션**: 반응형 모바일 버거 메뉴 기능 탑재 및 주요 영역 이동 스크롤 지원
2. **Hero 섹션**: 서비스 대표 슬로건 및 실시간 이동 매칭 컨셉의 Mockup UI 탑재
3. **Service Overview**: OMNI 플랫폼이 해결하고자 하는 교통 허브 및 탄소 저감 가치 제안
4. **Mobility Options**: 카셰어링, 택시, 공공자전거, 전동 킥보드, 대중교통 연계 등 5대 이동 수단에 대한 카드형 UI 소개
5. **How It Works**: 지도 조회부터 반납 및 결제까지 이어지는 심플한 4단계 서비스 프로세스 안내
6. **Pricing Section**: 기본 무료제부터 스마트 통근형 요금제, 구독형 PASS 패키지, 기업 비즈니스 맞춤 제안 소개
7. **Benefits Section**: 실시간 최적화 알고리즘, 에코 마일리지 적립 등 서비스의 강점 정리
8. **Promo Banner & QR**: 앱 다운로드 연동을 위한 App Store / Play Store 링크와 visual QR Code 마크업
9. **Semantic HTML5 & Responsive Design**: 시맨틱 태그 구조 및 모바일/태블릿/데스크톱 대응 반응형 디자인 제공

## 📁 폴더 구조
```
mobility-service-website/
├── index.html       # 웹 서비스 메인 파일 (HTML/CSS 포함)
├── README.md        # 프로젝트 설명 문서
└── assets/
    └── images/      # 외부 이미지 자산 배치용 폴더 (현재는 내장 SVG 및 CSS 렌더링 대체)
```

## 🚀 실행 방법
이 프로젝트는 별도의 웹 서버나 빌드 도구 없이 브라우저에서 바로 실행할 수 있는 정적 HTML 웹페이지입니다.

1. 이 리포지토리를 클론하거나 파일을 다운로드합니다.
2. `mobility-service-website/index.html` 파일을 크롬(Chrome), 사파리(Safari), 엣지(Edge) 등 최신 브라우저로 실행(더블클릭)합니다.

## 🛠️ 기술 스택
- **HTML5**: 웹 문서 구조 및 검색엔진 최적화(SEO) 대응 시맨틱 마크업 설계
- **CSS3 (Vanilla)**: 반응형 레이아웃 구현(Flexbox/Grid), 글라스모피즘 스타일, 트랜지션 효과 설계
- **SVG**: 이미지 파일 로드 의존성을 없애고 웹 로딩 속도를 최대화하기 위한 인라인 벡터 그래픽스 활용

## ⚠️ 제약사항 및 규칙
- **외부 CSS 미사용**: 가볍고 유지보수가 쉽도록 모든 스타일을 `index.html` 내부 `<style>` 태그 안에 직접 작성하였습니다.
- **외부 CDN 미사용**: 타사 라이브러리(Bootstrap, Tailwind, FontAwesome 등) 및 외부 이미지 리소스를 제거하여 네트워크가 단절된 오프라인 환경이나 GitHub Pages 호스팅 시 즉시 완벽히 렌더링됩니다.
- **JavaScript 최소화**: 성능 및 접근성을 보장하기 위해 순수 HTML과 CSS 선택자 구조만으로 반응형 토글 메뉴 및 레이아웃을 처리하였습니다.
